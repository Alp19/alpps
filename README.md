# alpps
Alp Port Scanner

Scans the ports of the target ip address one by one and saves the open ports in a txt file

Using:

<pre>
git clone https://github.com/Alp19/alpps.git
</pre>
<pre>
cd alpps 
</pre>
<pre>
bash alpps.sh or ./alpps.sh 
</pre>
